pub enum DataDomain<T: RegisterDomain> {
    Top(ByteSize),
    Pointer(PointerDomain<T>),
    Value(T),
}

An abstract domain representing either a pointer or a non-pointer value. Both non-pointer values and offsets of pointers are represented by the same abstract domain T.

Variants

Top(ByteSize)

The Top element of the domain. Describes a value for which nothing is known.

Pointer(PointerDomain<T>)

The value is a pointer to an abstract memory object.

Value(T)

The value is a non-pointer value or a pointer to global memory. The latter can happen if pointers to global memory are described by their absolute value.
