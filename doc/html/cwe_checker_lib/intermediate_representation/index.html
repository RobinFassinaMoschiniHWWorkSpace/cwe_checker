Module cwe_checker_lib::intermediate_representation[−][src]

This module defines the intermediate representation used to represent a binary and all its contained executable code.

The main data structure is the Project struct, which contains all information recovered about a binary during the disassembly step. To learn how individual instructions are encoded, you should first take a look at the Expression type and then at the Def and Jmp data types, which form the basis of the basic block Blk struct.
